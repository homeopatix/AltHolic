DESCRIPTION (Francais)
AltHolic est un petit plugin pour avoir une vision rapide de vos alts
Cliquer sur l'icon du petit sac de pieces pour affiche le porte-monnaie du personnage sélectionné
Clique sur l'icone de la classe, affiche l'equipement du personnage sélectionné
Clique sur le coffre, affiche le coffre du personnage sélectionné
Clique sur le sac à dos, affiche le sac à dos du personnage sélectionné
Clique sur le petit coffre en haut a gauche pour afficher le stockage partagé
Clique sur la loupe pour effectué une recherche
Clique sur l'icone du niveau pour afficher les statistiques
Clique sur l'icone de la race pour afficher les réputations
Clique sur le petit livre en haut a gauche pour afficher les quetes epiques des autres personnages
Clique sur le nom d'un personnage pour définir des informations sur ce personnage, survol d'un nom de personnage affiche les infos (uniquement si elle existe pour le personnage ciblé)
Avec le personnage CONNECTE cliquer sur un objet de la fenetre d'equipement vous permet
de définir le niveau de l'object, l'armure ou le dps, qui est visible par les autres perso
Presser la touche ALT pour déplacer l'icone
Presser la touche SHIFT et click sur l'icone AltHolic pour afficher la fenetre d'options
Vous pouvez cacher l'icone avec la commande /Alt icon ou la reafficher avec la meme commande
Permet de contrôler l'equipment de vos autres personnage même quand vous n'etes pas connecter avec celui-la
Vous pouvez cliquez sur la petite croix en haut a droite pour réduire la fenêtre ou sur le signe moins pour la réouvrir
Vous pouvez afficher l'icone de suppression de personnage dans la fenetre des options
Vous pouvez définir d'afficher les serveurs dans les options, et choisir quelle serveur vous voulez afficher
Vous pouvez cliquez sur l'icone de la flèche et vous pouvez voir et definir l'experience de chaque personnages
Vous pouvez afficher une barre au sommet de l'ecran qui affiche les icones principale
Cliquez sur la barre pour afficher sa fenetre d'options
Cliquez sur un serveur pour définir son nom pour le personnage sélectionné
Vous pouvez définir la vocation de votre personnage dans la fenêtre d'option
!!!! Attention si les personnages de vos différent serveur ont exactement le même nom cela ne fonctionne pas !!!!

Utilisation :
Cliquez sur la croix et entrer le nom de votre compte
Ensuite à chaque fois que vous vous connecter avec un personnage, le plugin enregistre les données et créé automatiquement
une nouvelle entrée, et sauvegarde le tous lorsque vous vous déconnecter


DESCRIPTION (English)
AltHolic is a small plugin to have a quick view of your alts
Click on the icon of the small bag of coins to display the wallet of the selected character
Click on the class icon, display the equipment of the selected character
Click on the chest, display the chest of the selected character
Click on the backpack, display the backpack of the selected character
Click on the small chest at the top left to display the shared storage
Click on the level icon to display the statistics
Click on the breed icon to display the reputations
Click on the small book at the top left to display the epic quests of the other characters
Click on the name of a character to define information about this character, hovering over a character name displays the information (only if it exists for the targeted character)
With the character CONNECTED clicking on an item in the equipment window allows you to
define the level of the object, armor or dps, which is visible to other characters
Click on the magnifying glass to search
Press the ALT key to move the icon
Press the SHIFT key and click on the icon AltHolic to display the options window
You can hide the icon with the command /Alt icon or redisplay it with the same command
Allows you to check your other character's equipment even when you are not connected with it
You can click on the small cross at the top right to minimize the window or on the minus sign to reopen it
You can display the delete character icon in the options window
You can set to display servers in the options, and choose which server you want to display
You can click on the arrow icon and you can see and set the experience of each character
You can display a bar at the top of the screen that displays the main icons
Click on the bar to display its options window
Click on a server to define its name for the selected character
You can set your character's vocation in the option window
!!!! Be careful if the characters from your different servers have exactly the same name, it doesn't work !!!!

Use :
Click on the cross and enter the name of your account
Then every time you connect with a character, the plugin saves the data and automatically creates
a new entry, and save it all when you log out


DESCRIPTION (Deutsche)
AltHolic ist ein kleines Plugin mit dem Sie schnell einen Uberblick uber Ihre Alts erhalten
Klicken Sie auf das Symbol des kleinen Münzbeutels, um die Brieftasche des ausgewählten Charakters anzuzeigen
Klicken Sie auf das Klassensymbol und zeigen Sie die Ausrüstung des ausgewählten Charakters an
Klicken Sie auf die Truhe und zeigen Sie die Truhe des ausgewählten Charakters an
Klicken Sie auf den Rucksack und zeigen Sie den Rucksack des ausgewählten Charakters an
Klicken Sie auf die kleine Truhe oben links, um den gemeinsam genutzten Speicher anzuzeigen
Klicken Sie auf das Ebenensymbol, um die Statistiken anzuzeigen
Klicken Sie auf das Rassesymbol, um den Ruf anzuzeigen
Klicke auf das kleine Buch oben links, um die epischen Quests der anderen Charaktere anzuzeigen
Klicken Sie auf den Namen eines Zeichens, um Informationen zu diesem Zeichen zu definieren. Wenn Sie den Mauszeiger über einen Zeichennamen halten, werden die Informationen angezeigt (nur wenn sie für das Zielzeichen vorhanden sind).
Wenn Sie mit dem Zeichen VERBUNDEN auf ein Objekt im Gerätefenster klicken, können Sie dies tun
Definieren Sie die Ebene des Objekts, der Rüstung oder des Schadenspunkts, die für andere Charaktere sichtbar ist
Klicken Sie auf die Lupe, um zu suchen
Drücken Sie die Alt-Taste, um das Symbol zu verschieben
Drücken Sie die UMSCHALTTASTE und klicken Sie auf das Symbol AltHolic , um das Optionsfenster anzuzeigen
Sie können das Symbol mit dem Befehls /Alt icon ausblenden oder mit demselben Befehl erneut anzeigen
Ermöglicht es Ihnen, die Ausrüstung Ihres anderen Charakters zu steuern, auch wenn Sie nicht damit verbunden sind
Sie können auf das kleine Kreuz oben rechts klicken, um das Fenster zu minimieren, oder auf das Minuszeichen, um es erneut zu öffnen
Sie können das Symbol zum Löschen von Zeichen im Optionsfenster anzeigen
Sie können die Anzeige von Servern in den Optionen festlegen und auswählen, welcher Server angezeigt werden soll
Sie können auf das Pfeilsymbol klicken und die Erfahrung jedes Charakters sehen und einstellen
Sie können oben auf dem Bildschirm eine Leiste anzeigen, die die Hauptsymbole anzeigt
Klicken Sie auf die Leiste, um das Optionsfenster anzuzeigen
Klicken Sie auf einen Server, um seinen Namen für den ausgewählten Charakter zu definieren
Du kannst die Berufung deines Charakters im Optionsfenster festlegen
!!!! Seien Sie vorsichtig, wenn die Charaktere von Ihren verschiedenen Servern genau den gleichen Namen haben, es funktioniert nicht !!!!

anzuzeigenVerwenden :

Jedes Mal, wenn Sie sich mit einem Charakter verbinden, speichert das Plugin die Daten und erstellt sie automatisch
einen neuen Eintrag und speichern Sie alles, wenn Sie sich abmelden
